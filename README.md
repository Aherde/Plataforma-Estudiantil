# Digitalización Inteligente y Conectividad Comunal - U.E.N.B. "Dr. Luis Padrino"

[cite_start]Este proyecto de Servicio Comunitario, realizado para la **Universidad Nacional Experimental de las Telecomunicaciones e Informática (UNETI)**, integra el desarrollo de un sistema de gestión escolar en la nube y la optimización de la infraestructura de red local[cite: 69, 71, 87].

## 🎯 Objetivo General
[cite_start]Diseñar e implementar una solución tecnológica integral que combine la digitalización administrativa mediante un sistema en la nube y la optimización de la infraestructura interna de red WiFi para mejorar la eficiencia académica y la estabilidad de la conectividad[cite: 85, 124, 125].

## 💻 Funcionalidades del Sistema (Módulo de Software)
[cite_start]Desarrollado con una arquitectura basada en la nube (Firebase), el sistema incluye las siguientes capacidades[cite: 111, 129]:

* [cite_start]**Gestión de Autenticación Segura:** Módulo de acceso restringido para garantizar la integridad del tráfico administrativo bajo protocolos de seguridad[cite: 129, 131].
* [cite_start]**Registro de Estudiantes:** Digitalización de los procesos manuales (inscripción y control) para evitar la pérdida o deterioro de la información física[cite: 93, 106].
* [cite_start]**Base de Datos en Tiempo Real:** Almacenamiento y gestión de datos académicos utilizando **Cloud Firestore**[cite: 111, 135].
* [cite_start]**Módulo de Reportes:** Visualización y generación de información académica digitalizada para el personal docente y administrativo[cite: 113, 129].
* [cite_start]**Respaldo en la Nube:** Garantía de disponibilidad remota y seguridad de los registros estudiantiles[cite: 112, 113].

## 🌐 Optimización de Infraestructura (Módulo de Red)
[cite_start]Además del software, se realizó una reingeniería de la red LAN para solucionar problemas de alta tasa de error de bits (BER) y microcortes[cite: 96, 114]:

* [cite_start]**Saneamiento de Cableado:** Limpieza física y fijación mecánica de cables UTP para eliminar fallos de conexión[cite: 130, 135].
* [cite_start]**Segmentación y Reubicación:** Reubicación estratégica de nodos y del router principal para maximizar la cobertura WiFi en Dirección, Administración y Aulas[cite: 85, 130, 135].
* [cite_start]**Seguridad de Red:** Implementación de protocolos WPA2/WPA3 para proteger la red interna del plantel[cite: 131].

## 🛠️ Tecnologías Utilizadas
* **Frontend:** React.js
* [cite_start]**Backend/Base de Datos:** Firebase (Firestore & Auth) [cite: 111, 129, 135]
* [cite_start]**Infraestructura:** Cableado estructurado UTP y protocolos de red inalámbrica [cite: 130, 131]
